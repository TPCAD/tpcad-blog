# Python 命名空间与作用域

「命名空间（Namespace）」是从名称到对象的映射。命名空间相互独立，不同命名空间允许有相同的名称。相同名称依靠其命名空间找到对应的对象。

一般有三种命名空间，按查找顺序分别为：

- 局部命名空间通常是函数或类内部区域
- 全局命名空间通常是当前 Python 模块
- 内置命名空间是 Python 内置，包含`abs`、`len`等内置函数。

命名空间的「作用域（Scope）」是 Python 程序中可以**直接访问**该命名空间的正文区域。直接访问指的是位于该文本区域的名称被非限定引用时，查找名称的范围包含该命名空间。

只有模块、类、函数（包括 lambda 表达式）才会引入新作用域。

在下方的程序中，函数`foo`引入了一个作用域和命名空间，当在该函数内部使用`bar`时，查找名称范围包括该命名空间，因此该命名空间的作用域就是函数`foo`内部。而在函数外部无法查找到`bar`，因为查找范围不包括函数`foo`引入的命名空间。

```python
def foo():
    bar = 42
```

Python 中一共有四种作用域，按照查找顺序分别为：

- Local：最内层，比如函数或方法内部
- Enclosing：嵌套函数的外层，包含非局部且非全局的变量
- Global：全局，即当前模块
- Built-in：Python 内置作用域

在函数中可以读取全局变量，若想修改全局变量，需要先用`global`关键字声明。

```python
num = 10
def inc():
    global num
    num += 1

inc()
assert(num == 11)
```

在嵌套函数内层中可以读取外层变量，若想修改外层变量，需要先用`nonlocal`关键字声明。

```python
def outer():
    num = 10
    def inc():
        nonlocal num
        num += 1
        return num

    return inc()

assert(outer() == 11)
```
